## 1. 环境准备

### 1.1 环境检查

当前项目运行环境配置：

| 配置项 | 当前值 | 说明 |
|--------|--------|------|
| 操作系统 | Windows | 从文件路径 `e:\xwechat_files\...` 识别 |
| 数据库端口 | 3308 | 在 application.yml 中配置 |
| 应用端口 | 8090 | 在 application.yml 中配置 |
| 数据库名 | cakeshop | 在 application.yml 中配置 |
| 数据库用户 | root | 在 application.yml 中配置 |
| 数据库密码 | 空 | 在 application.yml 中配置 |
| Java 版本 | 1.8+ | 在 pom.xml 中配置 |
| Maven 版本 | 3.8+ | 在 pom.xml 中配置 |

### 1.2 安装 MySQL

**Windows 用户安装步骤：**

1. 访问 [MySQL 官网](https://dev.mysql.com/downloads/installer/)
2. 下载 MySQL Installer for Windows
3. 双击安装包，选择 "Custom" 安装类型
4. 选择 MySQL Server 5.7+ 版本
5. **重要：配置端口为 3308**（与项目配置一致）
6. 设置 root 密码（项目当前配置为空密码）
7. 完成安装并启动 MySQL 服务

**验证安装：**
```bash
mysql -u root -P 3308
```

### 1.3 安装 Docker Desktop（可选）

如果需要使用 Docker 部署，安装 Docker Desktop：

1. 打开浏览器，访问 [Docker Desktop 官网](https://www.docker.com/products/docker-desktop/)
2. 点击 "Download for Windows" 按钮下载安装包
3. 双击下载的安装包，按照向导完成安装
4. 安装完成后，启动 Docker Desktop
5. 等待 Docker 引擎启动完成（右下角托盘图标变绿）

**验证安装：**
```bash
docker --version
docker-compose --version
```

## 3. 本地 Docker 部署

### 3.1 项目现有 Docker 配置

项目已包含完整的 Docker 配置文件：

#### Dockerfile（`Dockerfile`）

```dockerfile
FROM maven:3.8.6-openjdk-8 AS builder
WORKDIR /app
COPY pom.xml .
COPY src ./src
RUN mvn clean package -DskipTests

FROM openjdk:8-jre-alpine
WORKDIR /app
COPY --from=builder /app/target/*.jar app.jar
RUN apk add --no-cache tzdata && cp /usr/share/zoneinfo/Asia/Shanghai /etc/localtime
EXPOSE 8080
ENTRYPOINT ["java", "-jar", "app.jar"]
```

#### docker-compose.yml（`docker-compose.yml`）

```yaml
version: '3.8'

services:
  mysql:
    image: mysql:5.7
    container_name: cake_mysql
    restart: always
    environment:
      MYSQL_ROOT_PASSWORD: root
      MYSQL_DATABASE: cakeshop
      MYSQL_USER: admin
      MYSQL_PASSWORD: admin
    ports:
      - "3306:3306"
    volumes:
      - mysql_data:/var/lib/mysql
      - ./knight.sql:/docker-entrypoint-initdb.d/knight.sql

  cake-app:
    build:
      context: .
      dockerfile: Dockerfile
    container_name: cake_app
    restart: always
    environment:
      SPRING_DATASOURCE_URL: jdbc:mysql://mysql:3306/cakeshop?useUnicode=true&characterEncoding=utf-8&serverTimezone=Asia/Shanghai
      SPRING_DATASOURCE_USERNAME: admin
      SPRING_DATASOURCE_PASSWORD: admin
    ports:
      - "8080:8080"
    depends_on:
      - mysql
    links:
      - mysql

volumes:
  mysql_data:
```

## 4. 阿里云服务器部署

### 4.1 服务器准备

#### 4.1.1 购买云服务器 ECS

1. 登录阿里云控制台：https://ecs.console.aliyun.com/
2. 点击左侧菜单 "实例" -> "创建实例"
3. 配置实例：
   - **地域**：选择离你最近的区域（如华东1、华南1等）
   - **实例规格**：选择入门级配置（如 ecs.t6.large，2核4G）
   - **镜像**：选择 Ubuntu Server 20.04 LTS 64位
   - **存储**：系统盘选择 40GB SSD
   - **网络**：选择默认专有网络 VPC
   - **公网带宽**：选择按流量计费，带宽峰值 1-5 Mbps
   - **登录方式**：设置密码登录，记住密码！

4. 点击 "立即购买"，完成支付

#### 4.1.2 配置安全组

1. 在实例列表中找到你的服务器，点击实例名称进入详情页
2. 点击左侧菜单 "安全组"
3. 点击 "配置规则"
4. 添加以下规则：

| 规则 | 端口 | 来源 | 说明 |
|------|------|------|------|
| HTTP | 80 | 0.0.0.0/0 | Web 访问 |
| HTTPS | 443 | 0.0.0.0/0 | 安全访问（可选） |
| SSH | 22 | 0.0.0.0/0 | 远程连接 |
| 应用端口 | 8080 | 0.0.0.0/0 | 测试用 |
| MySQL | 3306 | 0.0.0.0/0 | 生产环境不建议开放 |

#### 4.1.3 连接服务器

**Windows 用户使用 PowerShell：**

```bash
ssh root@你的服务器公网IP
```

输入之前设置的密码，即可登录服务器。

### 4.2 安装 Docker 和 Docker Compose

登录服务器后执行以下命令：

```bash
# 更新系统
apt update && apt upgrade -y

# 安装 Docker
apt install -y docker.io

# 启动 Docker 服务
systemctl start docker
systemctl enable docker

# 安装 Docker Compose
apt install -y docker-compose

# 验证安装
docker --version
docker-compose --version
```

### 4.5 配置域名（可选）

#### 4.5.1 购买域名

1. 登录阿里云控制台：https://dns.console.aliyun.com/
2. 点击左侧菜单 "域名" -> "域名列表"
3. 点击 "注册域名"，搜索并购买你想要的域名

#### 4.5.2 配置域名解析

1. 在域名列表中找到你的域名，点击 "解析"
2. 添加解析记录：
   - **记录类型**：A
   - **主机记录**：www（或其他子域名）
   - **记录值**：你的服务器公网IP
   - **TTL**：默认 10 分钟

#### 4.5.3 配置 Nginx 反向代理

创建 `nginx.conf` 文件：

```nginx
server {
    listen 80;
    server_name your-domain.com;

    location / {
        proxy_pass http://localhost:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }
}
```

修改 `docker-compose.yml` 添加 Nginx 服务：

```yaml
services:
  nginx:
    image: nginx:latest
    container_name: cake_nginx
    restart: always
    ports:
      - "80:80"
      - "443:443"
    volumes:
      - ./nginx.conf:/etc/nginx/conf.d/default.conf
    depends_on:
      - cake-app
```

#### 4.5.4 配置 HTTPS（可选）

使用 Let's Encrypt 免费 SSL 证书：

```bash
# 安装 Certbot
apt install -y certbot python3-certbot-nginx

# 获取证书
certbot --nginx -d your-domain.com

# 自动续期测试
certbot renew --dry-run
```

## 5. 常见问题排查

### 5.1 数据库连接失败

**检查 MySQL 服务状态：**

```bash
# Windows
net start MySQL80

# Linux
systemctl status mysql
```

**检查数据库配置：**

```sql
mysql -u root -P 3308
SHOW DATABASES;
USE cakeshop;
SHOW TABLES;
```

**常见原因：**
- MySQL 服务未启动
- 端口配置错误（项目使用 3308，Docker 使用 3306）
- 数据库密码不匹配

### 5.2 应用启动失败

**查看应用日志：**

```bash
# 本地开发
mvn spring-boot:run

# Docker
docker-compose logs -f cake-app
```

**常见原因：**
- 数据库连接失败
- 端口被占用（8090 或 8080）
- JDK 版本不兼容（需要 JDK 1.8）

### 5.3 端口被占用

**Windows 查看端口占用：**

```bash
netstat -ano | findstr :8090
netstat -ano | findstr :8080
```

**结束占用端口的进程：**

```bash
taskkill /PID 进程ID /F
```

### 5.4 无法访问网站

**检查防火墙（Linux）：**

```bash
ufw status
ufw allow 80
ufw allow 8080
```

**检查安全组（阿里云）：**
- 确认 80 端口（或 8080 端口）已在安全组中开放

**检查容器运行状态：**

```bash
docker-compose ps
```

### 5.5 Docker 容器启动失败

**查看日志：**

```bash
docker-compose logs cake-app
docker-compose logs mysql
```

**常见原因：**
- 数据库初始化失败
- Maven 构建失败
- 网络连接问题
